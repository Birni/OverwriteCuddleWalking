Hook of APrimalDinoCharacter_UpdateBabyCuddling.

This function validates if the NewBabyCuddleType is walking and calls the original UpdateBabyCuddling function with modified parameters. The parameter NewBabyCuddleType will modify to food and the NewBabyCuddleFood will be filled by one random selected food of the FoodList. In case the NewBabyCuddleType is not walking this function calls the original UpdateBabyCuddling function without any modified parameters. In case of an error the original UpdateBabyCuddling function without any modified parameters.

Parameters
[in]_thisthe dino that gets updated
[in]NewBabyNextCuddleTimethe cuddle time (server time in seconds)
[in]NewBabyCuddleTypethe cuddletype (cuddle/food/walking) that the dinos requests
[in]NewBabyCuddleFoodthe food that the dino requests (if cuddletype is food)
